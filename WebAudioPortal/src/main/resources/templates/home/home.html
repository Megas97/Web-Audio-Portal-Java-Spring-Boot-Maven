I created this website as a university project in my third year studying Informatics at New Bulgarian University.
The goal of this project was to replicate all functions of a basic audio sharing service.
At the top you see the navigation bar with quite a few buttons in it. If you're on mobile then you'll see a sandwich button instead.

While to use the full functionality of the website you need to be registered and logged in, guest users still have the ability to view and download all public audio files, view the list with all registered users and their audio files and details as well as to view and use the shoutbox feature.

All registered and logged in users are allowed to upload mp3 audio files of up to 100 MB in size. Multiple file upload is also available but the size limit for all selected files to upload is again 100 MB. Only the uploader of a given mp3 audio file can delete, hide or unhide it at any given moment in time.

In case any of you are wondering how I built this site and what I used, I've put up a special tab in the navigation bar, which contains links to different topics that really helped me during the development of the website. Just remember - never give up! That's the golden rule of programming.

Of course, not all features can be covered in this page so I'll just let you explore the website on your own. It's more fun that way too :)
